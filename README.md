# LRP - Laboratuvar Yönetim Sistemi

Bu proje, üniversite laboratuvarlarındaki bilgisayar envanterini takip etmek ve öğrencilere donanım zimmetlemek için geliştirilmiş bir **ASP.NET Core Minimal API** projesidir.

## Özellikler
- **Admin Paneli:** Laboratuvar, bilgisayar ve öğrenci yönetimi.
- **Öğrenci Portalı:** Öğrencinin kendine atanan PC özelliklerini görmesi.
- **Otomatik Demirbaş Kodu:** Her lab için benzersiz `LABX-PC-XX` formatında kod üretimi.
- **SPA Mimarisi:** Sayfa yenilenmeden dinamik içerik yükleme.

## Nasıl Çalıştırılır?
1. Projeyi bilgisayarınıza indirin veya klonlayın.
2. Visual Studio ile `.sln` dosyasını açın.
3. Bağımlılıkların yüklenmesi için projeyi `Build` edin.
4. `F5` tuşuna basarak projeyi çalıştırın.
5. Veritabanı ve Admin hesabı (`erdem` / `7532159`) ilk çalıştırmada otomatik oluşturulacaktır
